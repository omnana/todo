# 待办事项应用

一款轻量、快捷、无干扰的待办事项管理应用，帮助用户轻松记录、组织和跟踪任务。

## 技术栈

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** - 样式框架
- **Zustand** - 状态管理
- **date-fns** - 日期处理
- **LocalStorage** - 本地数据存储

## 功能特性

### 核心功能
- ✅ **任务管理**：添加、编辑、删除、标记完成
- 🔍 **搜索功能**：实时搜索任务标题和描述
- 📊 **统计面板**：显示任务总数、进行中、已完成和完成率
- 🏷️ **分类管理**：预设分类（工作、学习、生活、购物、其他）+ 自定义分类
- 📱 **响应式设计**：适配移动端和桌面端

### 数据特性
- 💾 **本地存储**：所有数据保存在浏览器本地，无需登录
- 📤 **数据导出**：支持导出 JSON 格式备份
- 🔄 **数据同步**：自动保存，刷新页面不丢失数据

## 项目结构

```
src/
├── components/      # React组件
├── hooks/          # 自定义Hooks
├── stores/         # Zustand状态管理
├── types/          # TypeScript类型定义
├── utils/          # 工具函数
├── services/       # 数据服务（localStorage封装）
└── App.tsx         # 主应用组件
```

## 开发规范

- 使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释
- 网站的文本统一管理，方便之后拓展 i18n
- 确保移动端适配
- 保持设计简洁，不要过度设计

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 使用说明

1. **添加任务**：点击右下角的"+"按钮
2. **搜索任务**：在顶部搜索框输入关键词
3. **分类筛选**：点击分类标签筛选任务
4. **标记完成**：点击任务前的复选框
5. **编辑任务**：点击任务标题或描述进行编辑
6. **删除任务**：点击任务右侧的删除按钮
